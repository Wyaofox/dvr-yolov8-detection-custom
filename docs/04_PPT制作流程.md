### 第8页：我们的方案 — 重叠度检测

**标题：** Bbox-Polygon 重叠度检测

**内容：**
- 上方：算法思路
  1. 将禁区多边形渲染为像素mask
  2. 将检测框渲染为像素mask
  3. 计算两个mask的交集 / 检测框面积
  4. 重叠≥30% → 判定"人在区域内"
- 下方：示意图（禁区mask蓝色 + bbox mask绿色 + 交集红色）

**素材：**
- **截图：** 用PPT画3个重叠的矩形/多边形，用不同颜色填充

### 第9页：我们的方案 — 停留计时

**标题：** 停留计时器（Dwell Timer）

**内容：**
- 流程图：
  ```
  人进入禁区 → 记录时间T0 → 每帧检查是否仍在禁区内
     → 是：计算 (当前时间 - T0)
        → ≥3秒？→ 触发告警，标记已告警
        → <3秒？→ 继续等待
     → 否：清除计时，重置
  ```
- 关键参数：
  - `overlap_threshold = 0.3`（重叠度阈值）
  - `dwell_seconds = 3.0`（停留时间阈值）

### 第13页：代码讲解1 — 重叠度计算

**标题：** 关键代码：Bbox-Polygon重叠度计算

**内容：**
```python
def _box_polygon_overlap(box, polygon):
    x1, y1, x2, y2 = box
    box_area = max((x2-x1), 1) * max((y2-y1), 1)

    mask = np.zeros((h, w), dtype=np.uint8)
    cv2.fillPoly(mask, [polygon], 1)      # ← 渲染多边形

    box_mask = np.zeros_like(mask)
    box_mask[by1:by2, bx1:bx2] = 1        # ← 渲染bbox

    intersection = np.logical_and(mask, box_mask).sum()  # ← 交集
    return float(intersection) / float(box_area)         # ← 比例
```

**讲解标注：**
- 第4行：`fillPoly` — 把多边形画成mask，核心技巧
- 第7行：`logical_and` — 两个mask做与运算得到交集
- 第8行：返回值是0.0~1.0的比例

**素材：** 代码截图（带行号和标注箭头）
**动画：** 代码逐段高亮

**标题：** 关键代码：停留计时告警逻辑

**内容：**
```python
# 首次进入禁区 → 记录时间
if tid not in self.dwell_start[name]:
    self.dwell_start[name][tid] = now        # ← 记录进入时刻

dwell = now - self.dwell_start[name][tid]

# 停留超时 + 未告警过 → 触发
if dwell >= self.dwell_seconds \
   and (name, tid) not in self._alerted:     # ← 防重复
    self._alerted.add((name, tid))
    zone_alerts.append(...)

# 离开禁区 → 清除
for tid in left_tracks:
    del self.dwell_start[name][tid]           # ← 重置计时
    self._alerted.discard((name, tid))        # ← 允许再次告警
```

## 五、制作建议

1. **先录素材再做PPT** — 系统演示截图和视频是最真实的素材
2. **代码用截图不用文本框** — 截图带语法高亮，比PPT里手打代码好看
3. **动画不要太多** — 每页1-2个淡入动画即可，过度动画分散注意力
4. **字体要大** — 正文至少20号，代码至少16号，最后一排要看得清
5. **练习2-3遍** — 掐表练习，控制在20分钟内
