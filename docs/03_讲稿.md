传统的视频监控，需要保安盯着屏幕看，容易走神、容易遗漏。我们的系统让计算机自动完成这件事——实时检测画面中的人体目标，判断是否有人闯入禁区，自动告警并录制证据。

## 项目背景与需求（约2分钟）

【PPT翻页 → 需求分析】

先说背景。视频监控在安防、工地、仓库等场景非常普遍，但传统系统有几个痛点：

1. **人工成本高** — 需要专人24小时值守
2. **容易遗漏** — 人眼长时间盯屏幕会疲劳
3. **事后取证难** — 出了事翻录像效率低

我们的系统要实现的是：自动检测人体 → 判断是否闯入禁区 → 实时告警 → 自动录制。

需求可以概括为四个字：**准、快、稳、用**。
- **准**：不能乱报警（低误报）
- **快**：实时处理（>20 FPS）
- **稳**：长时间运行不崩溃
- **用**：操作简单，一键启动

## 技术方案与架构（约3分钟）

【PPT翻页 → 架构图】

整体架构是四层流水线：

**第一层：视频采集** — OpenCV通过USB摄像头实时采集视频帧，分辨率640×480

**第二层：目标检测** — 使用YOLOv11模型，加载到NVIDIA GPU上进行推理。这里用的是RTX 3050 Ti，单帧推理大概40毫秒

**第三层：区域检测** — 这是我们重点自研的部分。拿到检测结果后，判断每个人的检测框是否与预设禁区有足够重叠，并且停留了足够长的时间

**第四层：响应输出** — 三个出口：钉钉推送告警消息、自动录制10秒MP4视频、NiceGUI面板实时显示

## 核心算法详解（约5分钟）

【PPT翻页 → 算法对比】

原来的方案用的是**锚点触发**——判断人体的中心点、头顶、脚底等9个关键点，任何一个点落入禁区就报警。

这个方案有个严重问题：**人路过禁区边缘就误报**。你想，一个人正常走过通道，他的手臂可能刚好伸进禁区范围，系统就报警了。这在实际使用中是完全不可接受的。

【PPT翻页 → 我们的方案】

我们的方案分两步：

**第一步：Bbox-Polygon重叠度检测**

不再看点，而是看整个检测框。具体做法是：
- 用OpenCV的`fillPoly`函数，把禁区多边形画成一个像素级的mask
- 同时把检测框也画成一个mask
- 两个mask做`logical_and`运算，得到交集
- 交集面积除以检测框面积，就是重叠比例
- 重叠≥30%，才算"人在禁区内"

【指着代码】大家可以看这段代码，核心就是这个`logical_and`，非常高效。

**第二步：停留计时**

就算人在禁区内，我们也不马上报警。而是：
- 通过跟踪器给每个人分配一个唯一ID
- 记录这个人第一次进入禁区的时间
- 如果他连续停留超过3秒，才触发告警
- 如果他在3秒内离开了，计时归零，不算

【PPT翻页 → 优势对比】

对比一下效果：

| | 原方案 | 我们的方案 |
|---|---|---|
| 检测粒度 | 点级别 | 框级别 |
| 触发条件 | 瞬间 | 停留3秒 |
| 路过误报 | 频繁 | 几乎消除 |
| 配置灵活性 | 固定 | 重叠阈值和停留时间都可调 |

## 关键代码讲解（约3分钟）

【PPT翻页 → 代码页】

我选了两个最关键的代码片段来讲。

**代码段1：重叠度计算**（zone_manager.py 第22-55行）

```python
def _box_polygon_overlap(box, polygon):
    mask = np.zeros((h, w), dtype=np.uint8)
    cv2.fillPoly(mask, [polygon], 1)       # 渲染多边形mask
    box_mask[by1:by2, bx1:bx2] = 1          # 渲染bbox mask
    intersection = np.logical_and(mask, box_mask).sum()
    return float(intersection) / float(box_area)  # 返回重叠比例
```

核心思路就是**把几何问题转化为像素运算**。OpenCV的fillPoly非常高效，640×480的图不到1毫秒。

**代码段2：停留计时**（zone_manager.py 第180-225行）

```python
if tid not in self.dwell_start[name]:
    self.dwell_start[name][tid] = now       # 记录首次进入时间
dwell = now - self.dwell_start[name][tid]
if dwell >= self.dwell_seconds:              # 停留≥3秒
    self._alerted.add((name, tid))           # 标记已告警
    zone_alerts.append(...)
```

这里用字典`dwell_start`追踪每个人的进入时间，用集合`_alerted`防止重复告警。人离开后自动清除状态。

## 总结与创新点（约2分钟）

【PPT翻页 → 总结页】

总结一下我们这个项目的**三大创新点**：

**第一，算法创新** — bbox重叠度+停留计时，把区域入侵检测从"看一眼就报警"升级为"看三秒再报警"，大幅降低误报率。

**第三，本土化创新** — 钉钉告警集成，适配国内企业使用场景。

系统在 RTX 3050 Ti 上稳定运行于21-26 FPS，满足实时监控需求。
